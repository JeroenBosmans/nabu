Detailed Description

a named tuple class for a beam

the tuple fields are:

  • sequences: the sequences as a [batch_size x beam_width x max_steps] tensor
  • lengths: the length of the sequences as a [batch_size x beam_width] tensor
  • states: the state of the decoder as a possibly nested tuple of [batch_size x beam_width x state_dim] tensors
  • scores: the score of the beam element as a [batch_size x beam_width] tensor
